# Case Study 1 — FlyRank ML Internship

## Problem

Search teams often manage thousands of webpages, making it difficult to identify which pages deserve attention first. Simple rules cannot reliably prioritize pages because multiple search signals influence performance.

## What I Did

As part of the FlyRank Machine Learning Internship, I explored anonymized search data to understand relationships between search metrics and content performance.

I framed the business problem as a machine learning task, identified possible targets and evaluation metrics, performed exploratory analysis, and began building a repeatable workflow for content opportunity scoring.

Throughout the project, I focused on understanding the problem before selecting models, using careful, evidence-based reasoning instead of assumptions.

## Outcome

This internship is strengthening my understanding of practical machine learning workflows, feature engineering, data analysis, and decision-support systems using real-world search data.

# Case Study 2 — SafeMine: AI Smart Helmet

## Problem

Mining remains one of the most hazardous industries. Workers face risks such as toxic gases, unsafe environmental conditions, and delayed emergency response.

## What I Did

I contributed to the development of SafeMine, an AI-powered smart helmet designed to improve worker safety.

The system combines IoT sensors with machine learning to monitor environmental conditions, detect hazardous situations, and provide real-time alerts. During development I worked on integrating intelligent monitoring with a user-friendly software system that supports proactive safety decisions.

## Outcome

The project demonstrated how AI and IoT can work together to address real-world safety challenges while improving my experience in system integration and applied machine learning.

# Case Study 3 — AI Assignment Evaluator System

## Problem

Manual evaluation of student assignments requires significant time while maintaining consistent grading standards is difficult.

## What I Did

I developed an AI-powered assignment evaluation platform that automates grading and feedback generation.

The system processes uploaded assignments, evaluates responses using AI techniques, stores results using MongoDB, and presents detailed feedback through a Flask-based web application.

Throughout development I focused on building an end-to-end system rather than only implementing isolated AI models.

## Outcome

The project strengthened my skills in backend development, AI integration, database design, and building complete software solutions.

# Case Study 4 — ResuMatch

## Problem

Job seekers often struggle to understand how well their resumes match available opportunities, while recruiters spend considerable effort manually reviewing applications.

## What I Did

I built ResuMatch, an AI-powered job matching platform.

The platform parses resumes, extracts skills, compares them with job requirements, calculates similarity scores, and ranks candidates using machine learning techniques.

I designed the backend workflow, integrated resume analysis, implemented matching logic, and developed an interface that makes recommendations easier to understand.

## Outcome

ResuMatch demonstrates my ability to combine AI, backend engineering, databases, and user-focused design into a complete application.
