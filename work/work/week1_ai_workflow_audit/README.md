## About Me

I am Om Prakash Barmola, a final-year B.Tech Computer Science (AI & Data Science) student preparing for Software Engineering and AI internships. My weekly workflow includes coursework, DSA practice, AI/ML learning, hackathon projects, resume building, and interview preparation.

# Workflow Audit

| Task | Classification | Why |
|------|----------------|-----|
| Attend college lectures | Just Me | I need to actively participate and understand concepts myself. |
| Solve DSA problems | Collaborate with AI | AI can explain approaches and review my solution, but I must solve the problems myself. |
| Debug programming errors | Delegate to AI with Review | AI helps identify bugs quickly, but I verify every fix before using it. |
| Build AI projects | Collaborate with AI | AI assists with ideas and implementation, while I make technical decisions. |
| Study Machine Learning | Collaborate with AI | AI explains difficult concepts and quizzes me, but I practice independently. |
| Prepare for coding interviews | Collaborate with AI | AI conducts mock interviews and provides feedback. |
| Write documentation | Delegate to AI with Review | AI drafts documentation, but I edit it for accuracy. |
| Resume improvement | Delegate to AI with Review | AI suggests improvements while I ensure the content is truthful. |
| Research new technologies | Fully Automate | AI can summarize articles and documentation before I dive deeper where needed. |
| Plan weekly study schedule | Delegate to AI with Review | AI proposes a schedule that I adjust based on deadlines and priorities. |
| Take exams and assessments | Just Me | These must reflect my own understanding and academic integrity. |
| Maintain GitHub repositories | Collaborate with AI | AI suggests organization and README improvements while I control the repository. |

---

# Three Target Tasks

## 1. Debug Programming Errors

**Success Definition**

- Root cause identified.
- Bug fixed.
- Code runs without errors.
- I understand why the issue occurred.

---

## 2. Solve DSA Problems

**Success Definition**

- Solve independently first.
- AI reviews my approach.
- Final solution passes all test cases.
- I can explain the algorithm and complexity.

---

## 3. Build AI Projects

**Success Definition**

- Project requirements are clear.
- AI assists with architecture and implementation.
- Features work correctly.
- Code is documented.
- Project is deployed or demonstrated successfully.

---

# Reflection

This audit helped me identify which activities should remain my responsibility and where AI can improve my productivity without replacing my learning. I intend to use AI primarily as a collaborator and reviewer rather than as a replacement for critical thinking.
