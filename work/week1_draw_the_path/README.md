# Week 1 – Draw the Path

## Proof Statement

My primary claim is that I can build practical AI-powered software applications that solve real-world problems from idea to deployment.

The audience for this portfolio is a Software Engineering Hiring Manager evaluating candidates for internship or entry-level software engineering roles.

The single action I want them to take after reviewing my portfolio is to invite me for an interview.

---

## Why this Portfolio Exists

A resume and LinkedIn profile can list my skills and projects, but they cannot demonstrate how I approach problems, make technical decisions, and build complete software applications. My portfolio provides evidence of those abilities through real projects and case studies.

- The problem
