My primary claim is that I can build practical AI-powered software applications that solve real-world problems from idea to deployment.

The audience for this portfolio is a Software Engineering Hiring Manager evaluating internship or entry-level candidates.

The single action I want them to take is to invite me for an interview after reviewing my portfolio.
